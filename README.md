
# 📚 Smart Study Planner

## Project Overview

Smart Study Planner is a Python-based application that helps students prepare efficiently for exams. The tool automatically generates a personalized study plan based on subjects, syllabus size, and exam date. It distributes chapters across available days to help students manage time and track progress effectively.

---

## Problem Statement

Students often struggle to:

* Plan syllabus effectively
* Manage multiple subjects
* Track study progress
* Prepare before exams

This project solves these problems by automatically generating a structured study schedule.

---

## Features

* 📅 Automatic Study Plan Generation
* 📚 Multiple Subject Management
* ⏳ Exam Countdown
* 📊 Progress Visualization
* 🎯 Efficient Time Management

---

## Technologies Used

* Python

---

## How to Run the Project

### Step 1: Install Dependencies

```
pip install streamlit pandas
```

### Step 2: Run the Application

```
streamlit run smart_study_planner.py
```

### Step 3: Open in Browser

```
http://localhost:8501
```

---

## How It Works

1. Enter subjects and number of chapters
2. Select exam date
3. Click Generate Study Plan
4. View your personalized study schedule
